# SWATTED — An Awareness Site on Online Swatting

Final project for a Cybercrime course.

## What's in this repo

Six linked HTML pages plus a shared stylesheet:

- `index.html` — Tab 1: Home page (definition, overview, statistics)
- `swatter.html` — Tab 2: The Swatter (demographics, motives, methods)
- `victims.html` — Tab 3: Victim-Survivors
- `impact.html` — Tab 4: Crime Impact (on victims, families, society)
- `laws.html` — Tab 5: Laws (federal statutes, mens rea, actus reus)
- `help.html` — Tab 6: Help for Victim-Survivors (prevention, resources)
- `styles.css` — Shared stylesheet

## How to publish to GitHub Pages

1. Create a new public repository on GitHub.
2. Upload all files in this folder to the root of the repo (or drag and drop them into the GitHub web UI).
3. In the repo, go to **Settings → Pages**.
4. Under **Source**, select **Deploy from a branch**.
5. Under **Branch**, select `main` and `/ (root)`. Click **Save**.
6. Wait about one minute. Your site will appear at `https://<your-username>.github.io/<your-repo-name>/`.
